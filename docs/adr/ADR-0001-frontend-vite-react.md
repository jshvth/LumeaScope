# ADR-0001: Frontend with Vite + React

## Status

Accepted

## Context

We need a fast, lightweight frontend for upload, sidebar, and chat. The focus is fast iteration and simple Vercel deployment.

## Decision

We use Vite + React as the frontend base and Tailwind CSS for styling.

## Consequences

- fast dev startup
- simple build and deployment
- later migration to Next.js is possible but not planned
