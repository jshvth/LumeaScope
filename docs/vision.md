# Vision

LumeaScope makes long documents instantly searchable. The product feels like an expert who has read every page and can answer quickly and precisely with citations.

## Problem

- Manuals and contracts are long and hard to scan
- Users waste time searching for details
- Context is lost when pages are opened manually

## Solution

- Upload PDFs and technical manuals
- Automatic RAG index with vector search
- Chat with citations and page references

## Success Criteria

- Typical answers under 3 seconds
- Every answer includes at least one citation
- Users find relevant sections faster than manual search
